== Description ==

Blibahblubah Tag Cloud is a plugin that allows the blog owner to customize the tag cloud so that
any reader mousing over the tag cloud will see the words in the tag cloud flicker with
a customizable effect. This is just a fancy bell and whistle to add to your blog.

You can adjust the font growth, the highlight color, the font decoration (bold, underline,
italic, etc.), and bordering options.

== Installation ==
1. Copy the blibahblubah directory and its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your blibahblubah tag cloud effect in the Settings->Blibahblubah options page.

== Frequently Asked Questions ==

= What does blibahblubah mean? =

Bli-bah-blu-bah is the sound that you should make in your head as you are
mousing over your tag cloud. It is the sound of the font effects being
applied and unapplied as you move your mouse.
